# 在线素材库部署口令

> 将本地公共素材库整理为可在线访问的静态素材库时，可复制以下口令给 Codex 或 VS Code AI 助手。

## 准备静态素材库

```text
请基于我的公共素材库目录，整理为可部署的在线 mock 素材库。

要求：
1. 保持以下目录结构：
   - products/ 商品图片
   - avatars/ 头像图片
   - banners/ 横幅图片
   - receipts/ 单据票据
   - stores/ 门店现场
   - uploads/ 上传预览
   - empty/ 空状态图
   - icons/ 业务图标
2. 文件名使用英文、数字和短横线，不使用中文和空格。
3. 生成 manifest.json，结构参考 在线素材库/manifest.example.json。
4. 不上传真实客户隐私、手机号、身份证、合同原件等敏感素材。
5. 输出一份部署说明，包含素材库 baseUrl 和各分类示例 URL。
```

## 项目接入在线素材库

```text
请把当前原型接入在线 mock 素材库。

素材库 baseUrl：
<在这里填入你的在线素材库地址，例如 https://assets.example.com/mock-assets/>

要求：
1. 优先使用在线素材库 URL。
2. 保留本地 assets/images/mock/ 作为离线备用。
3. 图片引用必须符合 设计规范/03-演示素材规范.md。
4. 商品图、头像、Banner、票据、门店现场、上传预览不要混用。
```
