# 后台管理端 mock 图片素材包

本目录用于存放后台管理端原型的本地演示图片。

推荐分类：

```text
products/   商品图片：商品、物料、服务项目图片
avatars/    头像图片：用户、员工、客户头像
banners/    横幅图片：后台运营图、顶部宣传图
receipts/   单据票据：票据、合同、附件、单据预览
stores/     门店现场：门店、仓库、现场照片
uploads/    上传预览：上传组件预览图、图片附件
empty/      空状态图：暂无数据、空列表、无搜索结果
icons/      业务图标：功能入口图标、自定义业务图标
```

命名示例：

```text
products/product-001.jpg       商品图片 001
avatars/avatar-001.png         头像图片 001
receipts/receipt-001.jpg       单据票据 001
```

引用示例：

```html
<img src="./assets/images/mock/products/product-001.jpg" alt="商品图">
```
