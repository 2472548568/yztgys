# 后台管理端 PRD

本目录用于存放后台管理端 PRD，默认遵守：

- `设计规范/00-全局规范.md`
- `设计规范/01-后台管理端规范.md`

推荐结构：

```text
PRD/系统名称/后台管理端/
├── README.md
├── M01-模块名称/
│   └── F001-功能名称.md
├── 01-角色权限矩阵.md（可选）
└── 02-数据字典.md（可选）
```

跨端唯一功能基线统一维护在：

```text
PRD/系统名称/00-功能模块架构.md
```

推荐功能文档结构：

```text
PRD/系统名称/后台管理端/M01-模块名称/
    └── F001-功能名称.md
```

后台页面需明确列表字段、筛选条件、表格操作、批量操作、权限控制、状态流转和异常反馈。
