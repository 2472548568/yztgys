# 后台操作手册

> 本目录用于存放基于后台 PRD 和后台原型生成的管理员手册、快速入门指南和截图素材。

## 目录结构

```text
交付物/操作手册/
└── 后台管理端/
    ├── images/screenshots/
    └── YYYYMMDD-系统名称-管理员手册-V1.0.md
```

## 输入来源

- `PRD/系统名称/00-功能模块架构.md`
- `PRD/系统名称/后台管理端/`
- `原型/后台管理端/系统名称/`
- `设计规范/00-全局规范.md`
- `设计规范/01-后台管理端规范.md`

## 生成原则

- 手册只描述管理员和业务人员可见操作，不写接口、数据库、代码实现。
- 界面元素用【】标注，例如：【新增】、【保存】、【提交】。
- 截图统一放在 `交付物/操作手册/后台管理端/images/screenshots/` 目录。
- 只生成后台管理端操作手册。
