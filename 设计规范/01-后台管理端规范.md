# 01-后台管理端规范

> 后台管理端固定使用 `Vue 3 + Ant Design Vue + Vite`。适用于 PC 后台、运营后台、管理控制台、配置中心等桌面端系统。

## 技术栈约束

| 项目 | 规范 |
|---|---|
| 前端框架 | Vue 3 |
| 构建工具 | Vite |
| UI 组件库 | Ant Design Vue |
| 图标 | Ant Design Vue 图标或项目统一图标 |
| 图表 | ECharts |

禁止项：

- 禁止使用 Vant 组件或 Vant 风格组件。
- 禁止使用移动端底部 TabBar 作为后台主导航。
- 禁止用移动端卡片流替代后台核心表格。
- 禁止在后台页面中使用 375px/390px 固定移动端画板。

## 画板与布局

| 项目 | 规范 |
|---|---|
| 设计稿宽度 | 1920px 优先，兼容 1440px 和 1280px |
| 最小内容宽度 | 1200px 或按业务复杂度调整 |
| 顶栏高度 | 48px |
| 左侧菜单宽度 | 190px - 210px |
| 页面背景 | #F5F7FA |
| 内容区边距 | 16px |
| 面板圆角 | 0px - 4px |
| 表格行高 | 48px - 56px，订单分组表格可到 64px |
| 表单项高度 | 32px |

典型布局：

- 深色顶部栏 + 白色左侧菜单 + 浅灰内容区。
- 顶部栏左侧为系统/店铺名称，中部为一级业务导航，右侧为通知、消息、店铺切换等全局操作。
- 左侧菜单按业务模块分组，支持展开/收起；当前二级菜单使用紫色文字高亮。
- 内容区优先使用筛选区、工具栏、表格、分页的稳定结构。
- 面包屑置于内容区顶部，页面标题位于面包屑下方，标题下直接进入标签、筛选或业务内容。

## 页面类型

### 工作台

- 展示核心指标、待办事项、快捷入口和异常提醒。
- 指标卡片应控制数量，突出可行动信息。

### 列表页

- 必须包含查询筛选、工具栏、表格、分页。
- 表格列需定义字段名、字段含义、格式、排序、筛选、操作。
- 批量操作需明确选择规则、禁用态、二次确认和结果反馈。
- 电商订单、采购单、售后单、结算单等复杂列表优先使用“分组表格”结构：分组标题行展示单号、时间、来源标签和右侧快捷操作；分组内容行展示商品、金额、会员/收货、物流/支付、状态、操作。
- 列表标签页位于筛选区上方，用于按状态快速切换，例如：全部、待支付、待发货、已发货、已取消、已完成。
- 筛选区为浅灰底面板，字段横向密集排列，支持两行布局；主按钮使用紫色描边或紫色实心，重置按钮使用灰色描边。

### 详情页

- 使用分组信息、状态流转、操作记录、关联数据。
- 复杂详情可使用 Tabs 分区。

### 表单页

- 简单表单可使用 Modal 或 Drawer。
- 长表单使用独立页面或分步骤表单。
- 必填项、校验规则、默认值、禁用态必须明确。

### 权限配置页

- 角色、菜单、按钮、数据范围需分层表达。
- 权限项需支持查看、新增、编辑、删除、导入、导出、审核等动作拆分。

## 组件规范

| 场景 | Ant Design Vue 组件 |
|---|---|
| 表格 | Table |
| 查询表单 | Form、Input、Select、DatePicker、Cascader |
| 操作按钮 | Button、Dropdown、Popconfirm |
| 状态展示 | Tag、Badge、Tooltip |
| 详情信息 | Descriptions、Tabs、Timeline |
| 弹窗编辑 | Modal、Drawer、Form |
| 数据反馈 | Message、Notification、Result、Empty、Spin |

## 样例图风格细则

### 顶部导航

- 顶栏背景使用 #071B2E，高度 48px。
- 顶栏文字使用白色或 70% 白色，当前一级导航加粗或高亮，非当前项降低透明度。
- 顶栏图标统一 16px - 18px 线性风格，右侧图标间距 20px 左右。

### 侧边菜单

- 侧边栏背景使用 #FFFFFF，宽度约 200px。
- 一级菜单标题使用 13px - 14px，二级菜单使用 12px - 13px。
- 当前菜单使用 #8B5CF6，不使用厚重背景；可配合 #F4EEFF 作为轻量选中底。
- 菜单分组之间保持 8px - 12px 间距，图标保持线性灰色。

### 筛选区

- 筛选区放在白色页面面板内的浅灰区域，背景 #F6F7FB，内边距 16px。
- 表单标签右对齐或紧贴输入框，输入框高度 32px，边框 #DDE3EC。
- 一行可容纳 4 - 6 个筛选条件；条件较多时使用两行，不折成大卡片。
- 搜索按钮使用紫色描边或紫色实心，重置按钮使用灰色描边。

### 表格与订单分组

- 表头背景 #F6F7FB，文字 #1F2933，字号 12px - 13px。
- 表格边框使用 #E8ECF2，尽量轻，不使用粗重网格线。
- 订单类表格每条订单由“订单标题行 + 商品明细行”组成，标题行背景 #F6F7FB。
- 商品图片固定 40px - 48px 方形，使用细边框，文字两行截断。
- 金额右对齐，状态居中或左对齐，操作列靠右展示紫色文字链接。
- 行内标签使用小尺寸描边或浅底，例如“店铺”“PC”使用青绿色。

### 按钮与链接

- 主操作优先紫色：#8B5CF6。
- 表格内操作以文字链接为主，颜色 #8B5CF6。
- 批量操作按钮在未选择数据时禁用，使用浅灰文字与边框。
- 危险操作保留二次确认，使用红色文字或红色确认按钮。

## 后台验收清单

- 是否使用后台布局：左侧菜单、顶部栏、内容区。
- 是否符合样例图风格：深色顶栏、白色侧栏、浅灰内容区、紫色操作强调、密集表格。
- 是否使用 Ant Design Vue 风格的表格、筛选、表单、弹窗、抽屉。
- 是否没有出现 Vant 风格底部 TabBar、移动端 Picker、移动端卡片主导航。
- 是否包含筛选、排序、分页、批量操作、空状态、异常状态。
- 是否明确角色权限、按钮权限和数据范围。
