# 设计规范

> 本文件是后台管理端设计规范入口。后续 PRD、HTML 原型、页面交互和交付物说明默认遵守后台管理端规范。

## 规范文件

| 文件 | 适用范围 | 技术栈 |
|---|---|---|
| `00-全局规范.md` | 品牌、色彩、字体、状态、文案、通用交互 | 不限定 |
| `01-后台管理端规范.md` | PC 后台、运营后台、管理控制台 | Vue 3 + Ant Design Vue + Vite |

## 使用原则

- 开始生成 PRD 或原型前，确认本项目为后台管理端。
- 后台管理端共同遵守 `00-全局规范.md` 和 `01-后台管理端规范.md`。
- 不引用 Vant 风格组件，不生成移动端页面。
- 后续后台原型视觉以 `assets/backend-style-reference.png` 为基准：深色顶栏、浅色侧栏、浅灰内容背景、紫色操作强调、密集表格订单管理风格。
- 如需调整视觉、组件或交互基线，应先更新后台规范文件，再生成或修改原型。

## 交付检查

- 是否只使用 Ant Design Vue 风格组件和桌面端后台布局。
- 是否包含列表、详情、表单、弹窗、抽屉、空状态、异常状态等可验证页面状态。
- 是否在 PRD 的页面映射中标注后台页面入口。
- 是否避免移动端 TabBar、Picker、Popup 等组件。
